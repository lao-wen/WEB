小米13 Ultra销量火爆，网上却出现大量“差评”，是真翻车了吗？

不得不说，今年国产旗舰机迎来全面崛起，先是华为P60系列一机难求，现在小米13Ultra也销量火爆，看来苹果在高端要有压力了。不过也正是因为国产旗舰的崛起，网上突然涌现出大量的负面评价，究竟是小米真翻车还是另有原因呢？

首先要说明一点，小米13Ultra确实不是一款完美手机，这一点哪怕当着雷军本人面我也不会改变观点。新机既没有玻璃版，也没有陶瓷版，没延续前几代“透明探索版”、“一体化陶瓷”那种炫酷质感，而且小米13Ultra最受欢迎白色素皮版本需要抢购，应该是低估了粉丝热情备货量不足，希望这些问题可以在打造下一代手机时尽量避免。

吐槽归吐槽，尽管小米13Ultra不是一款最完美的手机，这却是目前综合表现最强高端旗舰，至少跟竞争对手相比明显处于上风。此时网上突然惊现大量的负面评价，集中吐槽这款手机厚度、屏幕翻车，评价文案、时间出奇一致，很难不象背后是不是有谁在推动，有什么不可告人的目的。
